重点

1.jquery对象和DOM对象的互相转换

<div id="box"></div>

jquery对象==>dom对象

var aBox = $("#box")[0];
aBox.innerHTML = "可以使用原生js的dom操作方法";

dom对象==>jquery对象

var aBox = document.getElementById("box");
var $aBox = $(aBox);
$aBox.html("可以使用jquery的dom操作方法");

2.jQuery选择器

语法 举例 描述
#id选择器 $("#hello") 匹配出id为'hello'的元素集合
.class选择器 $(".aClass") 匹配出class名为'aClass'的元素集合
Element $("div") 匹配出页面中所有div的元素集合
层级选择器 $("div p span") 匹配出div下的p标签下的span标签的元素集合
属性选择器 $("input[name='newsletter']") 匹配出name名为'newsletter'的表单元素集合
:nth-child() 选择器 $("p:nth-child(3)") 选取属于其父元素的第三个子元素的每个p元素
表单选择器 $(":input") 匹配所有 input, textarea, select 和 button 元素
#jQuery选择器更详细的使用方法请参考

菜鸟教程：http://www.runoob.com/jquery/jquery-ref-selectors.html

3.jQuery事件

在 jQuery 中，大多数 DOM 事件都有一个等效的 jQuery 方法。

$("p").click(function(){
 // 动作触发后执行的代码!!
});

常用的事件有：click点击事件、focus得焦事件、blur失焦事件、change事件等

#jQuery事件更详细的使用方法请参考

菜鸟教程：http://www.runoob.com/jquery/jquery-ref-events.html

4.jQuery Ajax

AJAX 是与服务器交换数据的技术，它在不重载全部页面的情况下，实现了对部分网页的更新。

jQuery 提供多个与 AJAX 有关的方法。最常用的是$.ajax()方法

$.ajax({
        url:"demo_test.json",
        async:true,  //是否异步处理，默认true
        type:"POST",  //请求方式：GET/POST
        data:"name=admin&age=18",  //要发送到服务器的数据,可以写在这里，也可以用地址栏传参的形式拼接在URL后面
        //data:{"name":"admin","age":"18"},  //另一种写法
        dataType:"json",  //预期服务器相应的数据类型
        success:function(data){
            //...请求到的数据以及操作
        },
        error:function(){
            //...请求失败回调
        }
});
#jQuery Ajax更详细的使用方法请参考

$.ajax()：http://www.runoob.com/jquery/ajax-ajax.html

常见问题整理

1.深度克隆和浅度克隆
var o = {a: [1]};

浅拷贝

var o1 = $.extend({}, o);
console.log(o1.a === o.a);  // true

深拷贝

var o2 = $.extend(true, {}, o);
console.log(o2.a === o.a);  //false
2.jquery绑定的事件多次执行

当使用on绑定的事件多次加载后（如写在函数或者其他绑定事件之中）触发时会多次执行

解决办法：在每次绑定事件之前，对该事件解绑，也就是 $(".test-btn").off("click")

3.动态生成的标签绑定click等方法不生效

原因是动态生成的标签，jQuery选择器无法找到

解决办法：使用事件委托，即用on方法把事件委托到父元素上（此元素没有被动态生成）

动态生成的class为.subEle的标签的点击事件

$(".wapper").on("click",".subEle",function(){
             //...
})
