以 jquery 为基础的前端组件

架构：jQuery + Bootstrap3 + ES5

组件：RequireJS、DataTables、jsTree、Bootstrap-fileinput、Bootstrap-select、dialog、FileSaver.js、Font-Awesome、zip.js、等

jQuery：JavaScript基础库，其他组件必需的依赖库。当前维护版本3.x，需要IE9+支持，对于IE8等较旧的浏览器，需使用jQuery 1.12。

官网：https://jquery.com/

Bootstrap3：基于jQuery的构建响应式网站框架工具包，主要用于界面UI的构建。支持IE8-9，但支持不完整*。

官网：https://getbootstrap.com/docs/3.3

RequireJS：一个JavaScript文件和AMD模块加载器。

官网：http://requirejs.org/

DataTables：基于jQuery的高级表格插件。通过一系列的扩展和插件，提供逐步增强的功能，并且提供表内编辑功能。

官网：https://www.datatables.net

jsTree：基于jQuery的交互式树插件。

官网：https://www.jstree.com

bootstrap-fileinput：基于Bootstrap的增强型HTML5文件输入插件，可用于各种文件的文件预览，提供多种选择方式。

官网：http://plugins.krajee.com/file-input

Bootstrap-select：基于Bootstrap的增强型下拉选择插件。

官网：http://silviomoreto.github.io/bootstrap-select/

dialog：基于Bootstrap的弹窗插件。

官网：http://www.lhgdialog.com

FileSaver.js：是在客户端保存文件的解决方案，适合需要不经外部服务器生成文件的web程序。支持IE10+。

官网：https://github.com/eligrey/FileSaver.js

Font-Awesome：一套绝佳的字体图标库，支持SVG图标。

官网：https://fontawesome.com

zip.js：为web端生成和读取大型zip文件提供的API。支持IE10+。

官网：http://gildas-lormeau.github.io/zip.js

以 React 为基础的前端组件

架构：React + Redux + Webpack + ES6/7

组件：Immutable、Axios、Lodash、math.js、Moment.js、pubsub-js、jQuery、FileSaver.js、zip.js、antd、Bootstrap4、Handsontable、DataTables、jsTree、Bootstrap-fileinput、Bootstrap-select、dialog、Font-Awesome、markdown-it等

React：用于构建用户界面的JavaScript库。React设计思想独特，性能出众，代码逻辑简单，属于革命性创新，从最早的UI引擎逐渐转变成一整套Web App的解决方案，是未来一段时间内Web开发的主流工具（包含但不限于React、Redux全家桶）。

官网：https://reactjs.org

Redux：是JavaScript应用程序的可预测状态容器，与React搭配使用时，用于状态管理。

官网：https://redux.js.org

Webpack：前端模块化的解决方案，核心是一个用于现代JavaScript应用程序的静态模块打包器。

官网：https://webpack.js.org

Immutable：不可变数据结构的实现，和React是绝配。

官网：http://facebook.github.io/immutable-js

Axios：基于promise的HTTP库，简单易用，功能强大。

官网：https://github.com/axios/axios

Lodash：是一个一致性、模块化、高性能的JavaScript实用工具库。

官网：https://lodash.com

math.js：包含广泛的数学库。

官网：http://mathjs.org

Moment.js：日期处理类库。

官网：http://momentjs.com

pubsub-js：基于发布/订阅设计模式的JavaScript通信库。

官网：https://github.com/mroderick/PubSubJS

jQuery：作为类库或其他三方依赖库使用。不推荐，逐步取代。

Bootstrap4：Bootstrap最新版，有重大改进，技术栈过渡时首选。默认使用flexbox弹性盒布局，支持IE10+。

官网：https://getbootstrap.com/

antd*：基于React实现的组件库。（待研究。在React架构下逐渐替代Bootstrap）。

官网：https://ant.design

Handsontable：基于JavaScript实现的外观像Excel的电子表格组件。

官网：https://handsontable.com

markdown-it：Markdown格式解析器。

官网：https://github.com//markdown-it/markdown-it
