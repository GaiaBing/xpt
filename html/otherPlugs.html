Font-Awesome 字体图标库

重点

Font Awesome提供了可缩放的矢量图标，您可以使用CSS所提供的所有特性对它们进行更改，包括：大小、颜色、阴影或者其它任何支持的效果。

在不下载文件情况下，将如下代码插入html 中：

<link href="//netdna.bootstrapcdn.com/font-awesome/4.7.0/css/font-awesome.min.css" rel="stylesheet">

Font Awesome图标几乎在任何地方都可以使用，只需要使用CSS前缀 fa ，再加上图标名称。 Font Awesome是为使用内联元素而设计的。我们通常更喜欢使用 <i> ，因为它更简洁。 但实际上使用 才能更加语义化。

1.基本图形
fa-camera-retro 
 <i class="fa fa-camera-retro"></i> fa-camera-retro
2.图形大小

使用 fa-lg (33%递增)、fa-2x、 fa-3x、<fa-4x，或者 fa-5x 类 来放大图标。

fa-lg fa-2x fa-3x fa-4x fa-5x 
<i class="fa fa-camera-retro fa-lg"></i> fa-lg
<i class="fa fa-camera-retro fa-2x"></i> fa-2x
<i class="fa fa-camera-retro fa-3x"></i> fa-3x
<i class="fa fa-camera-retro fa-4x"></i> fa-4x
<i class="fa fa-camera-retro fa-5x"></i> fa-5x
3.固定宽度

使用 fa-fw 可以将图标设置为一个固定宽度。主要用于不同宽度图标无法对齐的情况。 尤其在列表或导航时起到重要作用。

4.固定宽度

使用 fa-ul 和 fa-li 便可以简单的将无序列表的默认符号替换掉。

FileSaver.js文件保存的插件

重点

有时我们需要在浏览器上生成文件并保存到本地，这个借助 FileSaver.js 就可以很方便地实现。

1.安装配置

1.首先到其 GitHub 主页上将插件下载到本地。https://github.com/eligrey/FileSaver.js

2.然后在需要使用的页面中将 JavaScript 文件引入。

<script src="path/FileSaver.js"/>
2.使用说明

1）保存文本

var blob = new Blob(["Hello, world!"], {type: "text/plain;charset=utf-8"});
saveAs(blob, "hello world.txt");//保存一个叫hello word，内容为Hello，world！的文本

2)保存文件

var file = new File(["Hello, world!"], "hello world.txt", {type: "text/plain;charset=utf-8"});
saveAs(file);

Layer弹窗

起步

layer可以独立使用，也可以通过Layui模块化使用。所以请按照你的实际需求来选择。 如果你只是单独想使用 layer，你可以去 layer 独立版本官网下载组件包。 你需要在你的页面引入jQuery1.8以上的任意版本，并引入layer.js。 如果你使用的是 layui，那么你直接在官网下载 layui 框架即可，无需引入 jQuery 和 layer.js， 但需要引入layui.css和layui.js

作为独立组件

//引入好layer.js后，直接用即可
<script src="layer.js"></script>
<script>
layer.msg('hello');
</script>

在layui中使用layer

layui.use('layer', function(){
  var layer = layui.layer;
  layer.msg('hello');
});

重点

常用属性

1.type - 基本层类型

类型：Number，默认：0

layer提供了5种层类型。可传入的值有： 0（信息框，默认）1（页面层）2（iframe层）3（加载层）4（tips层）。 若你采用layer.open({type: 1})方式调用，则type为必填项（信息框除外）

2.title - 标题

类型：String/Array/Boolean，默认：'信息'

若需要自定义标题区域样式，那么你可以title: ['文本', 'font-size:18px;']， 数组第二项可以写任意css样式；如果你不想显示标题栏，你可以title: false

3.content - 内容

content可传入的值是灵活多变的，不仅可以传入普通的html内容，还可以指定DOM，更可以随着type的不同而不同。

4.area - 宽高

在默认状态下，layer是宽高都自适应的，但当你只想定义宽度时，你可以area: '500px'，高度仍然是自适应的。 当你宽高都要定义时，你可以area: ['500px', '300px']

5.btn - 按钮

信息框模式时，btn默认是一个确认按钮，其它层类型则默认不显示，加载层和tips层则无效。当您只想自定义一个按钮时， 你可以btn: '我知道了'，当你要定义两个按钮时，你可以btn: ['yes', 'no']。当然，你也可以定义更多按钮， 比如：btn: ['按钮1', '按钮2', '按钮3', …]，按钮1的回调是yes， 而从按钮2开始，则回调为btn2: function(){}，以此类推。

示例代码
 //弹出一个页面层
  $('#test2').on('click', function(){
    layer.open({
        type: 1,
        area: ['600px', '360px'],
        shadeClose: true, //点击遮罩关闭
        content: '\<\div style="padding:20px;">自定义内容\<\/div>'//或者content:$("#id");
    });
  });

DataTablesjQuery表格插件

Datatables有一个强大的api，用来处理表格上的数据，你可以添加数据到已经存在的表格， 或者对已经存在的数据进行操作。 API旨在能够很好地操作表格中的数据。

API实例可以通过以下三种方式创建：

$( selector ).DataTable();
$( selector ).dataTable().api();
new $.fn.dataTable.Api( selector );
