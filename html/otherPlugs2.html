math.js广泛的数学库

起步

Math.js是JavaScript和Node.js的一个广泛的数学库。它具有灵活的表达式解析器，支持符号计算， 附带大量内置函数和常量，并提供了一个集成解决方案，可处理不同的数据类型. 与JavaScript内置的Math库兼容。

Math.js可以在浏览器中作为常规JavaScript文件加载，使用全局变量math在加载后访问该库：

<!DOCTYPE HTML>
<html>
    <head>
        <script src="math.js" type="text/javascript"></script>
    </head>
    <body>
        <script type="text/javascript">
            //使用 math.js 
            math.sqrt(-4); // 2i
        </script>
    </body>
</html>

也可以使用require.js 来加载math.js

参考网址

官方网站：http://mathjs.org

Moment.js日期处理类库

起步

兼容Windows 7上的IE8，IE9，Windows XP上的稳定Chrome，Mac上的Safari 10.8以及Linux上稳定的Firefox。

Moment.js可以在浏览器中作为常规JavaScript文件加载

<script src="moment.js"></script>
<script>
    moment().format();
</script>

也可以做使用require.js 来加载

require.config({
    paths: {
        "moment": "path/to/moment",
    }
});
define(["moment"], function (moment) {
    moment().format();
});

获取日期

moment().format();//当前时间，例：2018-04-04T15:25:52+08:00
moment().format("MM/DD/YYYY");//修改时间格式：04/04/2018
moment().format('YYYY-MM-DD HH:mm:ss');//2018-04-04 15:37:20
moment().format('dddd');//Wednesday
moment().format('d');3
......

PubSubJS基于发布/订阅设计模式的JavaScript通信库

起步

PubSubJS具有同步解耦，因此主题是异步发布的。这有助于保持程序的可预测性，因为消费者在处理主题时不会 阻止主题的发起者。PubSubJS旨在用于单个进程内，并且不适用于多进程应用程序（如Node.js -具有多个子进 程的群集）。 

// 创建一个函数来订阅主题
var mySubscriber = function (msg, data) {
    console.log( msg, data );
};

// 将函数添加到特定主题的订阅方列表中。
// 我们保留返回的标记，以便能够取消订阅
// 之后的主题
var token = PubSub.subscribe('MY TOPIC', mySubscriber);

// 异步发布主题
PubSub.publish('MY TOPIC', 'hello world!');

// 同步发布主题，在某些环境中速度更快，
// 但是当一个主题触发新的主题时会变得混乱
PubSub.publishSync('MY TOPIC', 'hello world!');

取消订阅

var mySubscriber = function (msg, data) {
    console.log(msg, data);
};
var token = PubSub.subscribe('MY TOPIC', mySubscriber);
PubSub.unsubscribe(token);

官方网站：https://github.com/mroderick/PubSubJS
