重点

1.组件和props

组件使你可以将UI划分为一个一个独立、可复用的小部件，并可以对每个部件进行单独的设计。组件名称总是以大写字母开始。

1). 分类：函数式组件和类组件

函数式组件

function Welcome (props){
    return <h1>Hello, {props.name}</h1>
}

类组件

class Welcome extends React.Component{
    render(){
        return <h1>Hello, {props.name}</h1>
    }
}

比较：函数式组件更简洁；类定义的组件使用ES6写法，有局部状态

2). ReactDOM.render() 渲染组件

参数1：组件，参数2：父节点对象

3). props不可修改

所有React组件都必须是纯函数，并禁止修改自身props

注意：render中只能有一个根标签

2.state和生命周期

1). ES5和ES6在state写法上的区别参考上面网址中的写法对照表

在constructor()中初始state

用setState()改变state

setState可以接收一个函数，这个函数接受两个参数，第一个参数表示上一个状态值，第二个参数表示当前props

this.setState((prevState, props) => ({
  counter: prevState.counter + props.increment
}));

注意：直接修改state不会重新渲染

2). React 组件生命周期

用于在渲染dom前后或者移除dom前后插入事件

有三个状态：Mounting（已插入真实 DOM）、Updating（正在被重新渲染）、Unmounting（已移出真实 DOM）

注意：React事件使用驼峰命名，而不是全部小写

3.使用第三方框架

react通过webpack调用第三方框架

参考网址：https://www.jianshu.com/p/c8a805145046

export和import

ES6 在语言标准的层面上，实现了模块功能，方便模块的引入

export命令用于规定模块的对外接口，import命令用于输入其他模块提供的功能。

1). export的使用方法：

//写法一
export var firstName = 'Michael';
export var lastName = 'Jackson';
export var year = 1958;

//写法二（推荐）
var firstName = 'Michael';
var lastName = 'Jackson';
var year = 1958;

export {firstName, lastName, year};

//可以使用as关键字重命名
function v1() { ... }
function v2() { ... }

export {
  v1 as streamV1,
  v2 as streamV2,
  v2 as streamLatestVersion
};

2). import的使用方法：

import {firstName, lastName, year} from './profile.js';
//如果想为输入的变量重新取一个名字，import命令要使用as关键字，将输入的变量重命名。
import { lastName as surname } from './profile.js';

import后面的from指定模块文件的位置，可以是相对路径，也可以是绝对路径，.js后缀可以省略。

3). export default 命令：

用export default命令，可以为模块指定默认输出

// export-default.js
export default function () {
  console.log('foo');
}

其他模块加载该模块时，import命令可以为该匿名函数指定任意名字。

// import-default.js
import customName from './export-default';
customName(); // 'foo'

import _ from 'lodash';

4.使用create-react-app创建react开发环境

打开命令行

//全局安装create-react-app，如果安装过，跳过这一步
$ cnpm install -g create-react-app
//创建文件名为my-app的react项目
$ create-react-app my-app
//进入my-app文件夹
$ cd my-app/
//启动项目
$ npm start 或 $ yarn start

//使用yarn安装模块
$ yarn add module

然后耐心地等待一会儿，浏览器显示Welcome to React之后就成功了

打包项目：npm run build
