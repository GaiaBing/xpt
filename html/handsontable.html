handsontableJavaScript编写的电子表格组件

起步

Handsontable是一个用JavaScript编写的电子表格组件，不受任何外部框架的限制。它可以很容易地修改或扩展 自定义插件。Handsontable还绑定到使用JSON格式的任何源并处理大量数据。您可以轻松执行所有 CRUD操作， 并为最终用户提供类似Excel的体验。

IE:9,10,11;；火狐50+；谷歌56+

在使用前，我们先安装handsontable，可以使用npm，输入一下命令：

npm install handsontable

在你要使用handsontable的HTML文件中插入以下代码：

<script src="/dist/handsontable.full.js"></script>
<link rel="stylesheet" media="screen" href="/dist/handsontable.full.css">

重点

初始化一个hansontable

//添加一个<div>将变成电子表格的空元素。让我们给这个元素一个ID。
<div id="example"></div>
//初始化
var data = [
  ["", "Ford", "Tesla", "Toyota", "Honda"],
  ["2017", 10, 11, 12, 13],
  ["2018", 20, 11, 14, 13],
  ["2019", 30, 15, 12, 13]
];

var container = document.getElementById('example');
var hot = new Handsontable(container, {
  data: data,
  rowHeaders: true,
  colHeaders: true
});

常用方法

1.addHookOnce （键，回调）

将侦听器添加到指定的钩子名称,添加只使用一次的方法，用完后自动删除

hot.addHookOnce('beforeInit', myCallback);
2.alter（action，index，amount，source，keepEmptyRows）

允许通过插入/删除行或插入/删除列来更改表结构:

在给定的行上方插入新行index。如果索引是null或undefined，则新行将 在最后一行之后添加。

var hot = new Handsontable(document.getElementById('example'));
hot.alter('insert_row', 10);

删除给定的行index/列index。

var hot = new Handsontable(document.getElementById('example'));
//删除指定行
hot.alter('remove_row', 10);
//删除指定列
hot.alter('remove_col', 10);
3.clear()
清空表格数据

4.getCell （row,col,topmost）
根据行列索引，获取一个被渲染的单元格，如果该单元格未被渲染则返回null

5.getCellEditor(row,col)
通过提供的参数row和col参数返回单元格编辑器。

6.getCellMetaAtRow（row）
根据索引从单元元数组中返回一行。

7.getColWidth（col）
返回请求列的宽度。

8.getData(row1,col1,row2,col2)
获取指定范围的单元格的数据

9.getRowHeight（row）
返回请求行的宽度。

10.getSelected（）
以数组形式返回当前选定单元格的索引[startRow, startCol, endRow, endCol]。 开始行和开始列是活动单元格的坐标（开始选择的位置）。
